# watsonassistant-nodered-gateway
Cloud Functions (OpenWhisk) based gateway to make programmatic calls from IBM Watson Assistant to Node-RED

Programmatic calls can be used in Watson Assistant to get real-time information from backend systems, display it or control the dialog flow. The general procedure based on IBM Clpoud Functions is described here: https://cloud.ibm.com/docs/services/assistant?topic=assistant-dialog-actions#dialog-actions
This solution described how to use Node-RED flows leveraging an IBM Cloud Functions gateway.
